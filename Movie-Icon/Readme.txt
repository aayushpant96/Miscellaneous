The Objective of the project is to Set thumbnails (folder icon) for all the provided list of movies.

Note:
* The name of the folder should be the movie name.
* The program will automatically get the .ico for the movie name and save it in the "Icons" Folder. (User Should create the folder first)

To run the project:

* Run the project by passing two command line arguments: path-to-Main-Movies-Folder path-to-icon-folder

For e.g: If you export as jar file, the the command to run the jar file would be:

java -jar JarFilName D:\Java\Movies D:\Java\Movies\Icons

Note: Movies should contain subdirectories of all the movies.
